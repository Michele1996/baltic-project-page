BALTIC

A Benchmark for 3D Reconstruction Across Air and Underwater Domains Under Varying Illumination

Cross-domain reconstruction under air and underwater conditions.

Overview

BALTIC is a benchmark for evaluating 3D reconstruction methods under strong domain shifts between air and underwater environments with varying illumination. The dataset enables systematic analysis of how scattering, absorption, and lighting degradation affect reconstruction quality and trajectory accuracy.

Dataset

The dataset consists of 13 multi-view sequences (E1–E13) captured across different environments and lighting conditions, along with synchronized ground-truth tracking data.

Pipeline

The benchmark follows a standardized reconstruction-to-evaluation pipeline:

  1. Capture multi-view images for each sequence (E1–E13)
  2. Run COLMAP to recover camera poses and sparse 3D structure
  3. Extract reconstructed trajectories (E*_xyz_q.txt)
  4. Load ground-truth tracking data (E*_tf.txt)
  5. Align trajectories using similarity transform (Umeyama)
  6. Evaluate geometry and trajectory accuracy using the devkit

This ensures consistent and reproducible evaluation across all methods.

Devkit

We provide a lightweight evaluation toolkit for geometry and trajectory benchmarking. It operates directly on COLMAP reconstructions and ground-truth trajectories. 

git clone https://github.com/Michele1996/baltic-project-page.git
cd baltic-project-page

python chamfer.py      # geometry metrics
python alignment.py    # trajectory metrics (ATE, RPE)

Data format

Reconstruction: E*_xyz_q.txt

qw qx qy qz x y z

Ground truth: E*_tf.txt

timestamp | Translation + Rotation

The devkit computes Chamfer Distance, surface roughness, and trajectory metrics (ATE, RPE) after Umeyama alignment.

💻 Code & Evaluation

Benchmark

  • Task: Multi-view 3D reconstruction
  • Goal: Robustness under air–water domain shift
  • Metrics: Chamfer Distance, Surface Roughness, ATE, RPE

All methods are evaluated using identical reconstruction and alignment procedures.

Results

COLMAP reconstructions across domains. Underwater scenes exhibit reduced density and fragmentation due to scattering and absorption.

Reproducibility

  • Run COLMAP per sequence (E1–E13)
  • Run geometry evaluation (chamfer.py)
  • Run trajectory evaluation (alignment.py)

All results in the paper can be reproduced using the provided dataset and devkit.
